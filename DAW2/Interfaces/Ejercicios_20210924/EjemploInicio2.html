Ejemplo de la etiqueta blockquote

El elemento BLOCKQUOTE se usa para escribir una cita textual o un párrafo exacto y que este se diferencie del resto del texto. El atributo cite, define la URL del documento o mensaje original. Da información sobre la fuente donde se extrajo la cita.
